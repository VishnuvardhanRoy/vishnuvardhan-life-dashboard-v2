# Contributing to VISHNUVARDHAN Life Dashboard

Thank you for your interest in contributing! This guide will help you get started.

## Getting Started

1. Fork the repository
2. Clone your fork: `git clone https://github.com/YOUR_USERNAME/vishnuvardhan-life-dashboard-v2.git`
3. Create a feature branch: `git checkout -b feature/your-feature-name`
4. Install dependencies: `npm install`
5. Start development: `npm run dev`

## Development Workflow

### Code Style
- Use ESLint and Prettier for code formatting
- Follow React best practices
- Use functional components and hooks
- Add PropTypes or TypeScript types

### Commit Messages
- Use clear, descriptive commit messages
- Use present tense ("Add feature" not "Added feature")
- Keep commits focused and atomic

### Testing
- Test your changes locally before submitting
- Ensure the app builds: `npm run build`
- Check responsive design on multiple devices

## Pull Request Process

1. Ensure your code follows the style guide
2. Add clear PR title and description
3. Reference related issues
4. Ensure all changes work in development
5. Request review from maintainers
